Проект задуман как шахматный движок, основанный на теории вероятностей - учитывающий отдельно вероятность выигрыша и ничьей в каждой позиции, и вычисляющий эти вероятности исходя из гипотезы степенного распределения (которое выбрано из соображений простоты формул) и формулы Байеса для эмпирической оценки. Пока ничего этого не реализовано, но для построения эмпирической оценки позиции была накоплена база данных партий, которой можно пользоваться отдельно от основного проекта. Все партии получены из открытых источников, так что никаких ограничений по авторским правам нет.

Для установки на компьютер базы данных дебютов проверьте наличие следующих пакетов: apache php mysql g++ dialog
Например, в системе Debian можно набрать
# apt-get install apache php5 mysql-client mysql-server g++ dialog

Рекомендуется создать отдельную базу данных и отдельного пользователя mysql, например, так:
$ mysql -uroot
mysql> create user `chessplayer`@`localhost` identified by <password>;
mysql> create database chess;
mysql> grant all on chess.* to `chessplayer`@`localhost`;

и затем введите (от имени root)
# make install
В процессе установки Вам будет предложено ввести параметры доступа к базе данных.

Справочник будет доступен через веб-интерфейс по адресу localhost/chess/openings.php

На данный момент, в веб-интерфейсе Вы будете видеть только варианты, их оценки, названия дебютов (по-английски) и сами партии, но без отображения позиции. В будущем планируется также реализовать отображение позиции на диаграмме.
